 # Leer registros de una Base de Datos

 TAREA.- Investiga cuál es la diferencia entre utilizar MYSQLI_BOTH, MYSQLI_NUM o MYSQLI_ASSOC, en la instrucción .mysqli_fetch_array

 ## Solución:
 MYSQLI_NUM: Buscará el numero de la fila.

 MYSQLI_ASSOC: Obtendrá una fila de resultado como un array asociativo.
 
 MYSQLI_BOTH: Creará un único array con los atributos de ambas dos.
